# Next.js + Rails 연습 프로젝트

이 프로젝트는 Next.js(프론트엔드)와 Ruby on Rails(백엔드)를 통합하는 연습용으로 만들어졌습니다.

## 구조

- **Frontend:** Next.js
- **Backend:** Ruby on Rails

## 목적

- Next.js와 Rails를 함께 사용하는 개발 환경을 실습합니다.
- 프론트엔드와 백엔드 분리형 아키텍처를 이해합니다.

## 시작 방법

1. Rails 서버 실행
2. Next.js 개발 서버 실행

## 기타

이 프로젝트는 개인 학습용으로 제작되었습니다.
